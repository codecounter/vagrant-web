vagrant-web
===========

This is a vagrant configuration useful for web develop.

Contains
========

- [CentOS 6.5 x64](https://atlas.hashicorp.com/puphpet/boxes/centos65-x64)
- php 5.6.17
- apache 2.4.18
- mysql 5.6.28
- nginx 1.0.15
- beanstalkd 1.9
- nodejs 0.10.36
  - npm
  - bower
  - grunt
  - gulp
- golang 1.5.2
- redis 2.4.10

Installation
============

```
git clone https://github.com/codecounter/vagrant-web
cd vagrant-web
vagrant up

```
